# Contributing

Thanks for helping improve OpenClaw + Skylight integrations.

## Before you push

```bash
bash scripts/publish-gates.sh
bash scripts/scrub-for-publish.sh
```

## Scope

In scope: Skylight frame automation, household audit/propose/apply, Nextcloud Talk + Mail helpers.

Out of scope: NC-GCS fleet ops, HPB patches, fleet QA skills.

## Pull requests

1. One logical change per PR
2. Update `CHANGELOG.md` under `[Unreleased]`
3. No real emails, room tokens, or home paths in committed files
4. Document new gates in `docs/GATES.md`
